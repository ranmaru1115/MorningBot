# 🌤️ LINE Weather Bot (日本全国対応版)

指定した地域の天気予報を取得して、LINEへプッシュ通知するシンプルなPythonスクリプトです。
お天気API（ライブドア天気互換API）を利用しています。

デフォルトでは **東京（`130010`）** の天気を取得する設定になっていますが、同梱のXMLファイル、または本README内のリストを参照して `AREA_CODE` を書き換えることで、日本全国どこの地域でも利用可能です。
